
| **函數名稱**                   | **功能說明**         |
| -------------------------- | ---------------- |
| `count($arr)`              | 計算陣列中的元素個數       |
| `array_push($arr, $val)`   | 在陣列末尾新增一個或多個元素   |
| `array_merge($a1, $a2)`    | 合併兩個或多個陣列        |
| `in_array("search", $arr)` | 檢查陣列中是否存在某個值     |
| `array_keys($arr)`         | 取得陣列中所有的鍵名 (Key) |

### 如何遍歷陣列？

最常用的方式是 `foreach` 迴圈，它專門為陣列設計：

```
$colors = ["Red", "Green", "Blue"];
foreach ($colors as $value) {
    echo "顏色是：$value <br>";
}

// 如果是關聯陣列，可以同時取得鍵與值
foreach ($age as $name => $val) {
    echo "$name 目前 $val 歲 <br>";
}
```
